# Project: Modern C++ Topics

Practice/examples using C++ 11 to 20.

## Usage

1. Pre-requisites:

    Software:
      
    ```text
    Ubuntu 22.04
    Git client software.
    ```
    
2. Build:

    Building:
    
    Navigate to project home directory.

    cd bin
    
    The following scripts are available.
    
    ```bash
    ./c - compile gradle project

    ./r - run example
    ```
